Installer - PHP

Il existe 2 solutions pour installer PHP.

  • Depuis le site officiel (https://www.php.net/
  • Avec un logiciel (ex : Wampserver, Laragon, Easyphp)

Dans cette documentation, les logiciels ne seront pas utilisés car ils rajoutent des fonctionnalités dont on n’a pas besoin.

Pour son installation, il faut se rendre sur le site officiel et télécharger la dernière version stable.

Il y a 4 versions différente :

  • 2x 64bits : Non Thread Safe et Thread Safe
  • 2x 32bits : Non Thread Safe et Thread Safe

La version a téléchargé est celle qui correspond à ton système d’exploitation.

La différence entre Non Thread Safe et Thread Safe, c’est si on veut installer PHP pour un serveur Web. PHP est utilisé localement, donc on choisit Non Thread Safe.

Une fois que le fichier Zip est téléchargé, il faut le décompresser dans n’importe quel dossier. Le choix du dossier conseillé est dans « C:\Program Files\php\php-X.X.X ». Remplacer les X par la version téléchargée.

En créant un dossier par version, cela permet de pouvoir tester plusieurs versions différentes de PHP.

Pour que Windows reconnaisse PHP, il faut modifier les variables d’environnement.

Cliquer sur « Variables d’environnement… ».

Modifier la variable « Path ».

Créer une nouvelle saisie en ajoutant le chemin du dossier php-X.X.X.

Désormais Windows reconnait PHP. Pour vérifier que c’est vrai, il faut ouvrir l’invite de commande (CMD) et taper « php -v ».

Si une erreur survient à l’exécution de la commande, il faudra télécharger « Visual C++ Redistributable ».
