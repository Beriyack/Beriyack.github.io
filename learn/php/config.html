configuration - PHP

Ajouter une extension

Par défaut PHP ne possède pas toute la configuration nécessaire pour un développement optimale.

Pour ajouter une extension, exemple PDO qui permet de se connecter à une base de données.

Se rendre dans le dossier de « php-X.X.X ».

Copier les fichier « php.ini-development » sur le bureau.

Renommer-le en php.ini.

Ouvrir le fichier « php.ini » avec votre éditeur de texte préféré.

Modifier les lignes ci-dessous, pour qu’elles ne soient plus en commentaire.

  • « extension_dir = "ext" ». Cette ligne permet de donner le nom du dossier des extensions.
  • « extension=pdo_mysql » : Cette ligne permet d’activer l’extension PDO avec MySQL.
  • « extension=pdo_sqlite » : Cette ligne permet d’activer l’extension PDO avec SQLite.

Sauvegarder et déplacer le fichier « php.ini » dans le dossier « php-X.X.X ».

Désormais l’extension PDO pour MySQL et pour SQLite sont disponible.
