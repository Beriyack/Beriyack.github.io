Démarrer le serveur

Démarrer le serveur PHP en suivant ces instructions :

  1. Ouvrir l’invite de commande (CMD).
  2. Se rendre depuis le CMD dans le dossier de votre projet.
  3. Taper la commande « php -S localhost:8000 ».
    • -S : Démarrer un serveur
    • localhost:8000 : accessibilité du serveur qu’en local en utilisant le port 8000 (généralement pas utilisé par Windows)

Le serveur a démarré. Ctrl + C permet de stopper le serveur. C’est possible de mettre une adresse IP.
